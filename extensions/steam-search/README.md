# Steam Search

Search for games and DLCs on the Steam Store directly from Raycast. View prices, reviews, system requirements, and open them in your browser or directly in the Steam Desktop Client.

## Features

- **Quick Search:** Find any game or software available on Steam.
- **Detailed View:** See comprehensive information including:
  - Current price
  - Release date
  - Developer and Publisher info
  - Genres
  - Community recommendations/reviews
- **System Requirements:** View minimum PC requirements directly in the app.
- **Deep Linking:** Choice to open the store page in your default web browser or jump straight into the Steam Client.
- **SteamDB Integration:** Quick link to view advanced statistics on SteamDB.

## How to use

1. Open Raycast and type "Search Games".
2. Enter the name of the game you are looking for.
3. Press `Enter` to see full details.
4. Use the Action Panel (`Command + K`) to toggle between opening in Browser or the Steam Client.
